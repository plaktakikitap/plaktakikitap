# Ses Dosyaları

Bu klasöre aşağıdaki ses dosyalarını ekleyin:

| Dosya | Açıklama | Önerilen süre |
|-------|----------|---------------|
| `record-choice.mp3` | Plak seçimi — vintage cızırtı + lo-fi/caz melodisi | ~1.5 sn |
| `book-choice.mp3` | Kitap seçimi — kitap kapağı kapanma / sayfa çevirme | ~0.3–0.5 sn |
| `paper-flip.mp3` | Ajanda sayfa çevirme — kağıt hışırtısı | ~0.2–0.4 sn |
| `metallic-click.mp3` | Ataşlı sayfa çevrildiğinde — metalik tıkırtı | ~0.1–0.2 sn |

**Format:** MP3 veya WebM (tarayıcı uyumluluğu için MP3 tercih edilir).  
**Boyut:** Mümkün olduğunca küçük tutun (örn. 128 kbps, kısa süre).

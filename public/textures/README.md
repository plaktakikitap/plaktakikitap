# Defter dokuları (opsiyonel)

Bu klasöre aşağıdaki görseller eklenebilir; yoksa bile uygulama CSS ile placeholder doku kullanır.

- **leather.jpg** — Kapak için deri/sert cilt dokusu (sol panel). Önerilen: düşük kontrast, koyu kahverengi/gri ton.
- **paper.jpg** — Kağıt dokusu (sayfa alanı). Önerilen: hafif sarımsı, çok düşük kontrast noktacıklı doku.

Her iki dosya da yoksa gradient + SVG fractal noise ile varsayılan görünüm uygulanır.
